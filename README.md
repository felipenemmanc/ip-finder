# IPCLI - ip finder

#### FIND THE IP  
    example: ./ipcli ip --host google.com  
    example RESPONSE:  
    192.168.0.1  
    192.168.0.2  

#### FIND THE NAME SERVER  
    example ./ipcli server --host google.com  
    example RESPONSE:  
    ns.google.com  
    ns10.google.com
